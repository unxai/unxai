# UNXAI

> 未知无限 · 探索AI (Unknown · Unlimited)

一个基于现代Web技术构建的AI主题展示平台，融合3D神经网络可视化与沉浸式用户体验，展现人工智能领域的创新与探索。

## 📋 项目简介

UNXAI是一个开源的AI主题展示页面，通过前沿的Web技术呈现出极具未来感的用户界面。该项目不仅仅是一个静态展示，还集成了GitHub API，能够动态展示组织的项目和团队成员信息。

## ✨ 特性

- 🎨 **玻璃态UI设计** - 采用现代的玻璃拟态效果，创造层次感和深度
- 🌐 **3D神经网络可视化** - 使用Three.js构建的动态神经网络背景，随鼠标交互响应
- 📊 **GitHub集成** - 自动从GitHub获取项目和团队成员信息
- 📱 **响应式设计** - 完美适配从移动设备到桌面的各种屏幕尺寸
- ⚡ **高性能动画** - 基于GSAP的流畅动画效果，优化用户体验
- 🔧 **系统终端模拟** - 实时显示系统状态和操作日志
- 🌈 **渐变色与光效** - 精心设计的色彩方案，营造科技感氛围

## 🛠️ 技术栈

- **前端核心**: HTML5, CSS3, JavaScript
- **3D渲染**: Three.js
- **动画引擎**: GSAP
- **字体**: Inter, JetBrains Mono, Outfit
